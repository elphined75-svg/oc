# OC World Simulator v2.0

자캐 생성, 사진 등록, 관계도, 감정/호감도/신뢰/긴장도, 자동 대화, 시간 진행, 랜덤 사건, 브라우저 저장을 포함한 정적 웹앱입니다.

## GitHub Pages 업로드

1. 이 폴더 안의 `index.html`, `css`, `js`, `README.md`를 GitHub 저장소에 업로드합니다.
2. Settings > Pages에서 Deploy from a branch, main, /root를 선택합니다.
3. 배포 링크를 엽니다.

## 저장

같은 기기와 같은 브라우저에서는 자동 저장됩니다. 다른 기기로 옮기려면 `저장파일 내보내기`를 누르고 JSON 파일을 가져가서 `불러오기` 하세요.
